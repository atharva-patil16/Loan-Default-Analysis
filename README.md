📊 Loan Default Analysis Dashboard

1. Project Title / Headline

Loan Default Analysis: Risk, Demographics & Financial Insights Dashboard
An interactive Power BI dashboard designed to analyze loan defaults, customer demographics, and financial risk patterns to support data-driven lending decisions.

2. Short Description / Purpose

This dashboard provides a comprehensive view of loan distribution, borrower profiles, and default trends. It helps identify high-risk segments and uncover patterns across income, employment, credit score, and time.



## 3. Tech Stack

The dashboard was built using the following tools and technologies:

* 🐍 Python – Data processing and analysis
* 📂 Power Query – Data cleaning and transformation
* 🧠 DAX (Data Analysis Expressions) – Calculated measures 
* 📊 Power BI Desktop – Data visualization and dashboard creation
* 🔗 Data Modeling – Relationships between loan, customer, and financial datasets
* 📁 File Format – '.pbix' (development), '.png' (preview)




4. Data Source

 Dataset Type: Loan customer dataset (synthetic/financial dataset)

 Contents Include:

  * Loan amount and purpose
  * Customer demographics (age, marital status, education)
  * Employment type and income
  * Credit score categories
  * Default status and yearly trends

5. Features / Highlights

• Business Problem

Financial institutions face challenges in identifying loan default risks due to complex interactions between customer demographics, income levels, and credit behavior.

Key questions include:

* Which customer segments have the highest default rates?
* How do income and employment affect loan repayment?
* What trends exist in defaults over time?


• Goal of the Dashboard

 * Analyze loan distribution across different categories
* Identify high-risk borrower segments
 * Track default trends over time
 * Enable better credit risk assessment and decision-making


• Walkthrough of Key Visuals

🔹 Loan Overview

* Loan Amount by Purpose – Shows how loans are distributed across categories like home, business, and education
* Average Loan by Age Group – Highlights borrowing patterns across age segments

#### 🔹 Customer & Income Insights

* Average Income by Employment Type – Compares earning patterns across employment categories
* Default Rate by Employment Type – Identifies which employment groups are more likely to default

🔹 Credit Risk Analysis

* Median Loan by Credit Score Category – Examines how loan size varies with creditworthiness
* Total Loan by Credit Score Bins – Shows distribution of loans across risk levels

🔹 Demographics

* Loan by Marital Status (High Credit Score) – Compares borrowing behavior across marital groups
* Loans by Education Level – Highlights how education correlates with borrowing

🔹 Time-Based Trends

* Default Rate by Year – Tracks changes in default behavior over time
* YoY Loan Growth (%) – Measures yearly loan growth
* YoY Default Growth (%) – Identifies risk trend shifts

🔹 Advanced Insights

* Loan Distribution by Income Bracket & Employment – Breaks down total loan exposure across financial segments
* YTD Loan by Credit Score & Marital Status – Multi-dimensional risk segmentation



 • Business Impact & Insights

* Risk Identification:
  Unemployed and lower-income groups show higher default rates, indicating key high-risk segments

* Credit Strategy Optimization:
  Credit score segmentation helps in refining approval and interest rate strategies

* Portfolio Monitoring:
  Year-over-year trends allow financial institutions to track risk fluctuations

* *Customer Segmentation:
  Demographic insights (age, education, marital status) support targeted financial products

* Decision Support:
  Enables banks and analysts to make informed lending, pricing, and risk mitigation decisions
